steps to running of the Spring Boot Application:

How a Spring Boot application works (simple flow)

1️⃣ Main method runs

@SpringBootApplication
public class App {
    public static void main(String[] args) {
        SpringApplication.run(App.class, args);
    }
}

This starts everything.

2️⃣ Auto-Configuration starts

Spring Boot checks dependencies in pom.xml

Automatically configures:

Tomcat server

DispatcherServlet

Database configs

Beans

No manual setup needed.

3️⃣ IOC Container starts

Spring Framework creates all beans.

Injects dependencies using @Autowired.

4️⃣ Embedded Server runs

Built-in Tomcat starts (default port 8080).

App is ready to receive HTTP requests.

5️⃣ Request Flow
Client → Controller → Service → Repository → Database → Response

Example:

@RestController handles request

@Service contains logic

@Repository talks to DB

Response returned as JSON
